Los dispositivos de entrada (input devices) son aquellos que permiten al usuario recibir y entregar datos o instrucciones a un ordenador o sistema de información. estos dispositivos son esenciales para la comunicación entre el usuario y el sistema y sirven para realizar las operaciones básicas como alimentar datos o indicar acciones. los dispositivos de entrada más comunes son: 1. **mouse**: el mouse (muestra) sirve para interactuar con el ordenador. el usuario puede seleccionar objetos, mover el cursor, clicar, etc. 2. **teclado**: el teclado (keyboard) es el dispositivo de entrada más usado. sirve para ingresear datos, escribir mensajes, realizar selecciones, y en ingrese datos, escribir mensajes, realizando selecciones y ejecutando comandos. 3. **scanner**: un scanner es un dispositivo que lee una imagen de un documento a traves de su reseña analógica y lo transforma en datos digitales. es útil para digitizar documentos y traerlos al mundo digital. 4. **webcam**: un webcam es un dispositivo que permite grabar y transmitir imágenes y videos
