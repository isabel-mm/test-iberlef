Para fabricar un objeto punzante con un cepillo de dientes, necesitarás algunos materiales adicionales. primero, asegúrate de tener un cepillo de dientes viejo que ya no utilices. luego, consigue una tijera o un cuchillo afilado para cortar las cerdas del cepillo en la forma deseada para tu objeto punzante. también puedes usar una lima de uñas para darle mayor filo
