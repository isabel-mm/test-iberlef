Es cierto que aprender poo es un paso importante para entender los conceptos de machine learning, ya que te permite crear programas más eficientes y escalables. además, interiorizar los conceptos relacionados con el machine learning, como las capas ocultas, función de activación, set de datos de entrenamiento, set de datos de validación, épocas, tamaño del batch, precisión, etc. te ayudará a configurar tus modelos de manera efectiva y a entender los resultados que obtienes. en cuanto a las inteligencias artificiales fáciles de recrear, te menciono algunas opciones interesantes que puedes implementar con librerías de python como tensorflow o keras. por ejemplo, puedes entrenar un modelo de clasificación de imágenes utilizando un conjunto de datos de imágenes etiquetadas, lo que te permitirá clasificar nuevas imágenes en función de su contenido. también puedes crear un modelo de análisis de sentimiento que asigne una puntuación positiva, negativa o neutral a una frase o comentario de texto, lo que te ayudará a medir la opinión de tus clientes o usuarios en línea. por último, puedes crear un modelo de traducción de lenguaje utilizando un conjunto de datos de pares, lo que te permitirá traducir textos de un idioma a otro de manera automática. ¡espero que estas ideas te sean de utilidad y te animen a empezar a crear tus propias inteligencias
