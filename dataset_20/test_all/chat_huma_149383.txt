Los componentes generalmente vienen preensamblados por ser un equipo complejo. las partes principales son: 1. dispositivos internos 1.1 mainboard (es la placa que interconecta todos los componentes). 1.2 cpu (es el dispositivo que ejecuta las instrucciones y/o programas proporcionadas por el usuario). 1.3 ram (almacena programas y datos importantes para que el procesador pueda acceder inmediatamente a ellos). 1.4 fuente de poder (proporciona bajo demanda la energía que requieren los componentes). 1.5 gpu (permite generar una interfaz gráfica, puede estar integrada dentro del cpu o se puede añadir mediante un puerto de expansión). 2. dispositivos esenciales de entrada 2.1 teclado (permite introducir instrucciones que serán interpretadas por el procesador). 2.2 mouse "opcional" (útil en caso de disponer de una interfaz gráfica, pero no es indispensable). 3. dispositivos esenciales de salida 3.1 pantalla (permite visualizar la información proporcionada por el procesador). para construir el equipo literalmente desde cero habría que construir componentes extremadamente pequeños, como un procesador. actualmente, es una de las tareas más arduas. se recurre a una técnica llamada litografía para lograrlo, aunque se puede hacer un procesador rudimentario a partir de transistores, pero su
