Precalentar el horno y forrar una bandeja para hornear. para asegurarse de que el horno esté lo suficientemente caliente para hornear las papas fritas cuando estén listas, ajustar la temperatura a 400 grados fahrenheit (200 grados celsius) y permitir que se caliente por completo. luego, usar un trozo de papel pergamino para forrar una bandeja grande para las papas fritas. puede sustituir el papel pergamino por papel de aluminio si lo prefiere. es importante no colocar las papas fritas de rutabaga directamente en la bandeja para hornear, ya que los bordes se quemarán más
